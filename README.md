# Computer-Network-Project-Phase2

### Server

```bash
make
./server [port1]
```

### Client(console mode)

```bash
make
./console_client [server ip]:[port1] [port2]
```

### Client(web mode)

```bash
make
./client [server ip]:[port1] [port3]
#macOS
open http://[client ip]:[port3]/
#linux
xdg-open http://[client ip]:[port3]/
```
